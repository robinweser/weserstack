---
title: fallbackValue
---

A helper function to create a fallback value for a property.

## The Gist

```ts
import { fallbackValue } from '@weser/styling'

const minWidthFallback = fallbackValue('width', [
  '-webkit-min-content',
  'min-content',
])
```

## Parameters

| Parameter | Type            | Description                                                        |
| --------- | --------------- | ------------------------------------------------------------------ |
| property  | string          | The property to create a fallback for.                             |
| values    | `Array<string>` | The values to use for the fallback. Last one is the default.       |
| match     | string?         | What values is matched by the fallback. Default is the last value. |

### match

By default, the fallback is matched by the last value. You can provide a custom match value by passing a string.

```ts
const minWidthFallback = fallbackValue(
  'width',
  ['-webkit-min-content', 'min-content'],
  'min'
)

// this would now match the fallback
const style = {
  width: 'min',
}
```

## Returns

([T_Fallback](/styling/api/types/T_Fallback)) A fallback object that can be passed to the [createRenderer](/styling/api/createRenderer) function.
