---
title: unit
---

This plugin converts values to the given unit.<br />
It is aware of unitless properties such as `line-height`, zero-values and also adds units to multiple values inside an array.

## The Gist

```ts
import { unitPlugin } from '@weser/styling'

const plugin = unitPlugin('px', {
  fontSize: 'rem',
})
```

## Parameters

| Parameter          | Value                                                                                                                        | Default                                                                                                       | Description                                                           |
| ------------------ | ---------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------- | --------------------------------------------------------------------- |
| unit               | `ch` \| `em` \| `ex` \| `rem` \| `vh` \| `vw` \| `vmin` \| `vmax` \| `px` \| `cm` \| `mm` \| `in` \| `pc` \| `pt` \| `mozmm` | `px`                                                                                                          | The global unit that gets applied to all values.                      |
| unitPerProperty    | `Record<keyof React.CSSProperties, string>`                                                                                  | `{}`                                                                                                          | Property-specific unit overwrites.                                    |
| isUnitlessProperty | `(property: keyof React.CSSProperties) => boolean`                                                                           | [isUnitlessProperty](https://github.com/robinweser/css-in-js-utils/blob/master/modules/isUnitlessProperty.ts) | A function that determines whether a property should remain unitless. |

## Returns

([Plugin](/styling/api/plugins)) A plugin function that can be passed to the [createRenderer](/styling/api/createRenderer) function.
