---
title: unit
---

This plugin converts numeric values to the given unit.<br />
It is aware of unitless properties such as `line-height`, zero-values and also adds units to multiple values inside an array.

## The Gist

```ts
import { unitPlugin } from '@weser/style'

const plugin = unitPlugin('px', {
  fontSize: 'rem',
})
```

## Parameters

| Parameter          | Type                                     | Default                                                                                                       | Description                                                           |
| ------------------ | ---------------------------------------- | ------------------------------------------------------------------------------------------------------------- | --------------------------------------------------------------------- |
| defaultUnit        | `string`                                 | `'px'`                                                                                                        | The global unit that gets applied to all numeric values.              |
| propertyMap        | `Partial<Record<keyof T_Style, string>>` | `{}`                                                                                                          | Property-specific unit overwrites.                                    |
| isUnitlessProperty | `(property: string) => boolean`          | [isUnitlessProperty](https://github.com/robinweser/css-in-js-utils/blob/master/modules/isUnitlessProperty.ts) | A function that determines whether a property should remain unitless. |

## Returns

([Plugin](/style/api/plugins)) A plugin function that can be passed to the [createRenderer](/style/api/createRenderer) function.
