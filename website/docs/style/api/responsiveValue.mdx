---
title: responsiveValue
---

A function to cast a responsive array value to its expected type.<br />

<Callout title="Important Note">
  This requires the
  [responsiveValuePlugin](/style/api/plugins/responsiveValue) to be
  configured.
</Callout>

## The Gist

```ts
import { responsiveValue } from '@weser/style'

const style = responsiveValue(['blue', , 'red'])
```

## Parameters

| Parameter | Type       | Description                         |
| --------- | ---------- | ----------------------------------- |
| value     | `Array<T>` | The responsive array value to cast. |

## Returns

(`T`) The cast value.
