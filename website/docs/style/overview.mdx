A convenience layer and tool belt on top of [css-hooks](https://css-hooks.com) for React.
It aims to bring the [Fela](https://fela.js.org) ergonomics to CSS Hooks.<br />
This package works well in combination with the [theme](/theme/overview) and [keyframe](/keyframe/overview) packages.

<PackageBadges package="@weser/style" />

## Installation

```sh
# npm
npm i --save @weser/style

# yarn
yarn add @weser/style

# pnpm
pnpm add @weser/style
```

## Benefits

This packages combines the ergonomics of [Fela](https://fela.js.org) with the convenience of [css-hooks](https://css-hooks.com), including things like:

- Vendor Prefixing
- Responsive Values
- Fallbacks Values
- Custom Plugins
