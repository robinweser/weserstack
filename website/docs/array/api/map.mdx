---
title: map
---

Maps the array based on the provided mapper function.<br />
Similar to `Array.prototype.map`.

## The Gist

```ts
import { map } from '@weser/array'

const mappedArray = map([1, 2, 3], (value) => value * 2)

// => [2, 4, 6]
console.log(mappedArray)
```

## Parameters

| Parameter | Type                                                              | Description          |
| --------- | ----------------------------------------------------------------- | -------------------- |
| arr       | `Array<T>`                                                        | The array to map.    |
| mapper    | `(value: T, index: number, length: number, array: Array<T>) => R` | The mapper function. |

## Returns

(`Array<R>`) The mapped array.
