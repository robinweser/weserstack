Creates a new node with a unique id.

## The Gist

```tsx
import { create } from '@weser/tree'

const node = create({
  children: [],
})
```

## Parameters

| Parameter | Type                                               | Description         |
| --------- | -------------------------------------------------- | ------------------- |
| node      | [T_BaseNodeInput](/tree/api/types/T_BaseNodeInput) | The node to create. |

## Returns

(`T`) A new node with a unique id.
